# MusMux
MusMux (Music Multiplexer) is a music player without support for playlists. Why? Because I have found that I find clicking the next song myself more enjoyable than getting a random one that I may not be in the mood for. MusMux is specifically designed with that in mind, so that at the end of every song it has a little popup window appear and ask which of some random songs you want next. 

For building, I made this in Visual Studio Community 2026 using .NET 8, C#, and WinUI 3.

MusMux makes use of TagLib#, see THIRD-PARTY-NOTICES.txt for more info.
